Guide

VXP apps are MediaTek/MRE apps. All phones which run VXP files are MediaTek, but not all MediaTek phones can run VXP files.

Since there are many different bad websites with ads and generally bad UI’s, I’ve thought it was my obligation to make a simple website with no ads, mobile-safe version and generally an archive of this.

Installing VXP apps

This depends on the phone. For most phones, all you need is to copy it onto the phone/memory card. However, on Nokia phones, you will need a SIM card (active or inactive) and the Nokia Resing tool. Nokia phones will not launch unless the app is tied to the SIM’s IMSI (NOT IMEI!). The Nokia Resing tool does not have those limitations, so using the tool, you can:
- read the IMSI
- make a VXP runner (a VXP file which can open other VXP files)
- patch VXP files with your IMSI
- change installed apps (Weather, Opera Mini..) with custom VXP’s

Notes

Most of these VXP files are cheap shovelware. These apps were mainly made to sell via carriers and nothing else. Don’t expect quality!
